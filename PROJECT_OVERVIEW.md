# FPS RePlayer — プロジェクト概要（HP制作用資料）

> このファイルは紹介ホームページを制作するための素材・指針をまとめたものです。
> 別PCでの作業や、デザイナー／AIへの指示出しにそのまま使えます。

---

## 1. アプリ名と一言説明

**アプリ名：FPS RePlayer**

> FPSゲームの録画を「1コマずつ前後に行き来」して検証できる、Windows用の動画プレイヤー。
> 気になった瞬間はカーソル追従の虫めがねで拡大、9:16の縦型クリップとして書き出せる。

**キャッチコピー案**
- 「死んだ瞬間を、1コマ単位で見直す。」
- 「コマ送り・コマ戻し・拡大・縦型書き出し。検証のための動画プレイヤー。」
- "Frame-by-frame replay for FPS players."

**基本情報**
- 現行バージョン：v1.1.0
- 対応OS：Windows 10 / 11（64bit）
- 配布形態：無料・オープンソース（MITライセンス）
- GitHub：https://github.com/capycappy/FPS-RePlayer
- ダウンロード：GitHub Releases の `FPSRePlayer-vX.Y.Z-win64.zip`（解凍して exe をダブルクリック、インストール不要）

## 2. 主な機能

- **前後両方向のコマ送り** … 1フレーム単位で前進・後退（長押しで連続送り）
- **カーソル追従の虫めがね** … マウス位置をその場で拡大（最大20倍）。死亡シーンの相手位置や着弾を細かく確認
- **倍速・低速再生（0.1〜16倍）** … 音声も速度に追従して再生
- **音声波形タイムライン** … 銃声など大きい音がスパイクで一目で分かり、その瞬間にすぐ飛べる
- **サムネイル帯（フィルムストリップ）** … 動画全体をサムネイルで俯瞰し、見たい場面へ素早くシーク
- **IN / OUT 区間指定** … 検証・書き出ししたい範囲を指定
- **9:16 縦型クリップ書き出し** … 注目シーンを縦型動画として切り出し（SNS／ショート向け、H.264 + AAC）
- **前後ファイルへワンキー移動** … 同フォルダ内の録画を Ctrl+→ / Ctrl+← で次々検証
- **最近のファイル履歴** … 右クリックで直近10件をすぐ開く
- **キー＆マウス割り当てカスタマイズ** … 操作を自由に再割り当て
- **日本語／英語対応** … 起動時にOS言語を自動判定、設定で切替可
- **幅広いフォーマット対応** … MP4 / MKV / MOV / AVI / WebM ほか（内蔵FFmpeg、別途インストール不要）

### 開発のきっかけ（ストーリー）
市販プレイヤー（PotPlayer / GOM / VLC / KMPlayer）では「**前後両方向のコマ送り**」と
「**カーソル追従の部分拡大**」を1本で両立できるものが無かったため自作。
（タルコフ配信者の検証スタイルが発端）

## 3. ターゲットユーザー

- **FPSプレイヤー／競技プレイヤー** … 自分のプレイ（特に死亡シーン）をコマ単位で検証したい人
- **Escape from Tarkov などハードコアFPSのプレイヤー** … 「どこから撃たれたか」を1フレームずつ確認
- **ゲーム配信者・動画クリエイター** … 検証クリップや縦型ショートを手早く切り出したい人
- **動作・タイミング解析をしたい人** … スポーツ動作解析ソフト的な使い方も可能
- 一言でいうと：**「録画を“検証”したいゲーマー」**

## 4. 技術スタック

- **言語**：Python 3.12
- **GUIフレームワーク**：PySide6（Qt 6）
- **動画デコード／エンコード**：PyAV（FFmpeg 8.x を内蔵）
- **数値処理**：NumPy（波形・サムネイル生成）
- **音声出力**：Qt Multimedia（QAudioSink）
- **配布パッケージング**：PyInstaller（単体exe化、FFmpeg同梱）
- **ライセンス**：MIT

**設計上の特徴（技術アピール用）**
- フレーム後退は「キーフレームへシーク → 順方向デコード ＋ フレームキャッシュ」で実現（フレーム間圧縮に対応）
- 再生はバックグラウンドの先読みデコードスレッドで、音声クロックに同期。デコード負荷で映像がカクつかない設計
- 縦型書き出しは FFmpeg フィルタグラフ（crop → scale → pad）で歪み・黒帯なし

## 5. スクリーンショットを撮るべき画面の一覧

HPに載せると効果的なカット（★は優先度）：

1. ★★★ **メイン画面（オーバービュー）** … 動画＋下部のサムネイル帯＋波形タイムラインが見える全体像。アプリの第一印象
2. ★★★ **虫めがね拡大（Before / After）** … 拡大前と、カーソル位置を拡大している様子の2枚。目玉機能
3. ★★ **音声波形タイムライン**（拡大表示） … 銃声のスパイク＋IN/OUTの黄色区間＋赤いプレイヘッド
4. ★★ **縦型書き出しの操作中** … 画面に出た9:16枠を配置している様子
5. ★ **書き出した縦型クリップの例** … 実際の出力（before=元の横長／after=縦型）
6. ★ **右クリックメニュー** … 「最近のファイル」「次/前のファイル」が出ている様子
7. ★ **設定ダイアログ** … 言語切替＋キー/マウス割り当て（カスタマイズ性アピール）
8. （任意）**コマ送り連写のGIF/動画** … 前後にコマ送りしている動きはGIFや短い動画が最も伝わる

> 撮影のコツ：実際のゲーム録画（タルコフ等）を読み込んだ状態で撮ると説得力が高い。
> ウィンドウ単位は `Alt + PrintScreen`、範囲指定は `Win + Shift + S`。幅1200〜1600px推奨。

## 6. HP制作で訴求すべきポイント

**コアの訴求（これだけは外さない）**
- **「前後コマ送り × カーソル追従拡大」を1本で両立** … 市販プレイヤーにできない“検証特化”が最大の差別化
- **検証作業が速い** … 波形で音の瞬間へ即ジャンプ、Ctrl+→で次の録画へ、コマ送り長押しで連続確認

**安心・手軽さ**
- **インストール不要**（解凍してダブルクリックだけ）／**Python等の事前準備不要**
- **無料・オープンソース（MIT）** … コードが公開されていて安心
- **日本語対応**

**機能の幅**
- **9:16縦型書き出し** … 検証だけでなく、SNS/ショート動画の素材作りにも
- **幅広いフォーマット対応**（MP4/MKV等、FFmpeg内蔵）
- **操作のカスタマイズ**（キー＆マウス再割り当て）

**ターゲットに刺さる言葉**
- 「“どこから撃たれた？”を1フレームで突き止める」
- 「死亡リプレイ検証のための専用プレイヤー」

**導線（HPに置くべきボタン/リンク）**
- 「ダウンロード」→ GitHub Releases（`...win64.zip`）
- 「ソースコード / GitHub」→ リポジトリ
- 動作環境の明記：Windows 10/11 64bit

**注意書きとして添えると親切**
- 初回起動時にWindows SmartScreen警告が出る場合がある（未署名のため）→「詳細情報」→「実行」で起動できる旨
